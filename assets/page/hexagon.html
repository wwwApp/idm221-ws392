Hexagon Defense

  • Web Game
  • Summer 2018
  • Web Game Design
  • Visit the Site

Hexagon Defense is a simple defense game to test your reflexes as you need to prevent the Invader from going into the Hexagon Portal. The goal of the game was to provide players with visually pleasant interface and engaging game mechanics, using custom graphics.

Using Adobe Illustrators, I built few mock-up screens for the game ideas I came up with. Once the idea was approved to be feasible and meet the requirements, I extracted elements, such as hexagon portal and two balls, from the mock-up screens and created sprites for each of them. Then, I used Phaser.js to create actual game screens that load and interact with the game elements and that are connected into the complete game.
